This is my personal user space for QMK configs. See [QMK Userspace](https://github.com/qmk/qmk_userspace).

Right now it only contains my [Arsenik](https://github.com/OneDeadKey/arsenik) keymaps for the CSTC40 (rev3), an ortholinear keyboard with a [Planck MIT](https://40s.wiki/boards/planck) layout.

The keymaps were generated with the help of [Aekeynox](https://github.com/OneDeadKey/qmk-config-aekeynox).
